# 디자인 브리프

## 방향

키워드는 따뜻한 전문성이다. 심리치료 분야의 안정감과 기업교육 페이지의 검토 용이성을 함께 가져가야 한다. 그래서 감성적인 장식보다 정보 구조, 여백, 얇은 경계선, 실제 활동 이미지, 명확한 CTA를 우선한다.

## 참고 디자인

- Notion: 따뜻한 백색, 얇은 선, 정리된 정보 구조
- Apple: 첫 화면의 큰 메시지, 넓은 여백, 명확한 CTA
- Framer: 강한 다크 톤은 메인으로 쓰지 않고, 하단 Contact의 집중감에만 제한적으로 참고

## 컬러

| 역할 | 색상 | 의도 |
| --- | --- | --- |
| Ink | `#23211f` | 순검정보다 부드러운 본문색 |
| Paper | `#fffdf9` | 따뜻한 백색 배경 |
| Soft | `#f6f5f4` | 섹션 구분 배경 |
| Blue | `#1d6fb8` | 문의 CTA |
| Green | `#2a7a66` | 회복, 안정, 전문성 |
| Rose | `#b65b67` | 예술심리의 온기 |
| Gold | `#a97830` | 활동 날짜와 신뢰 근거 강조 |

## 타이포그래피

- 기본: Inter, Pretendard, Noto Sans KR, system sans-serif
- 제목은 크고 단단하게 쓰되 자간은 0으로 유지한다.
- 본문은 16px에서 18px 사이를 기준으로 두고, 기관 담당자가 빠르게 훑을 수 있도록 줄 간격을 넉넉하게 둔다.

## 레이아웃

- Hero: 이름과 핵심 메시지, CTA, 실제 프로필 이미지가 첫 화면에서 바로 보이게 구성
- Profile: 소개 문단과 핵심 정보 3개를 분리
- Programs: 대표 커리큘럼 4개를 반복 카드로 구성
- Proof: 최근 활동과 외부 신뢰 근거를 짧은 카드로 구성
- Curriculum: 담당자가 바로 비교할 수 있는 표 형태
- Contact: 어두운 집중 섹션으로 문의 행동을 명확히 유도

## 이미지 원칙

- 실제 프로필, 실제 교육 포스터, 실제 강의 이미지를 우선 사용한다.
- 블러, 추상 배경, 과도한 힐링 이미지 사용은 피한다.
- 최종 공개 전 블로그 이미지 사용 권한을 확인한다.
