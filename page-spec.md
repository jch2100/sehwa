# 구현용 페이지 설계서

## 파일 구성

- `index.html`: 완성형 정적 랜딩 페이지
- `styles.css`: 반응형 스타일, 색상 토큰, 섹션 레이아웃
- `brand-report.md`: 브랜딩 분석 리포트
- `content-inventory.md`: 블로그 콘텐츠 인벤토리
- `design-brief.md`: 디자인 브리프
- `page-spec.md`: 구현 설계서

## 섹션 순서

1. Header
   - 브랜드명: 조세화
   - 메뉴: 프로필, 프로그램, 신뢰 근거, 문의
2. Hero
   - 대상: 기업/기관 담당자
   - 메시지: 예술심리 기반 마음회복 프로그램
   - CTA: 교육 문의, 블로그 보기
   - 시각 자료: 조세화 프로필 사진
3. Brand Message
   - 블로그 프로필 문구를 웹 카피로 압축
4. Profile
   - 전문 영역, 주요 대상, 전환 목표
5. Programs
   - ART:REST, TCI 팀 이해, 부모교육, 집단상담
6. Proof
   - H은행, 서울노동권익센터, 한국강사신문, 칼럼
7. Curriculum
   - 프로그램별 대상, 목적, 기대 효과
8. Contact
   - 이메일, 블로그 링크, 문의 안내

## 반응형 기준

- 980px 이하: Hero와 소개 영역을 1열로 전환, 프로그램과 신뢰 카드는 2열
- 680px 이하: 모든 주요 그리드를 1열로 전환, 내비게이션은 숨김
- 긴 한국어 문장이 버튼이나 카드 밖으로 밀리지 않도록 `clamp`, `min-width: 0`, 넉넉한 패딩 사용

## 검수 기준

- 첫 화면에서 이름, 전문 분야, 문의 행동이 보일 것
- 최근 활동 10개 이상이 콘텐츠 인벤토리에 반영될 것
- 의료 효과를 확정적으로 말하지 않을 것
- 기업/기관 담당자가 프로그램 종류와 신뢰 근거를 30초 안에 확인할 수 있을 것
